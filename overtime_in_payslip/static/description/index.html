Overtime Request

DeeB

Overview

odoo App will provide Employee Overtime Management functionality. internally You can request many Overtime requests, Restrict employee overtime request, multiple employee load,overtime and payslip connected and many more features..

Feature

Overtime Request for Employee.
Notify To Hr Manager.
Approve Overtime By Hr Manager.
Set different Overtime Type.
Calculation based on overtime type like per day, hour.
Workflow Draft => Submit Request => Hr Approval => Done
Notify Employee when request is approved
Calculate approved overtime amount in payslip
Print employee overtime requests

How To Work

Employee Create overtime request
Create New Overtime Type
Approve Request Internally
Calculate Overtime In Payslip
Print Overtime Requests
